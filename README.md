# C++ Function Exercises

Collection of six introductory C++ exercises focused on decomposing problems into reusable functions.

## Contents

The exercises are stored independently under the Ficha 1 directory:

    ficha1.cpp/ex1.cpp
    ficha1.cpp/ex2.cpp
    ficha1.cpp/ex3.cpp
    ficha1.cpp/ex4.cpp
    ficha1.cpp/ex5.cpp
    ficha1.cpp/ex6.cpp

Each file contains its own entry point and should be compiled separately.

## Learning goals

- Declare and call functions
- Separate calculations from console interaction
- Work with parameters and return values
- Practice conditional logic and numerical operations
- Compile small C++ programs from the command line

## Requirements

- A compiler with C++17 support

## Compile and run

Example for the first exercise:

    g++ -std=c++17 ficha1.cpp/ex1.cpp -o ex1
    ./ex1

Use the equivalent command for the other exercise files.
